Git og Github

Git er et system der man kan laste opp filene sine til en slags «sky». Der kan du se endringene dine og gå tilbake til eldre versjoner av dokumentet ditt. Det gjør det enklere å samarbeide med andre ettersom man kan se på endringene de har gjort. Eller du kan enkelt jobbe på flere datamaskiner på en gang. Det er altså flere personer som kan jobbe på samme fil og dermed sette det sammen til en fil senere, men det er ikke alltid Git klarer å «merge» kodene sammen og du må dermed gjøre dette manuelt.

GitHub er en løsning der man kan lagre koden sin helt gratis. Dermed har du alltid en backup av filene dine. Her får du også versjonskontroll og du kan dele filene dine med andre. Versjonskontroll vil si at du kan se hvordan dokumentet ditt har utviklet seg og du kan når som helst gå tilbake til hvilken versjon av dokumentet du vil.
